Spotifinder Privacy Policy

Last Updated:

1. Introduction

Spotifinder ("we", "our", or "us") is a Chrome browser extension that allows users to search Spotify songs from highlighted text on webpages. This privacy policy explains how we handle your data when you use our extension.

2. Data Collection

Spotifinder collects and stores the following data:

  • Spotify OAuth Tokens: Access tokens required to authenticate with Spotify's API. These are stored locally in your browser using Chrome's secure storage system.
  • Selected Text: Text you highlight on webpages is temporarily stored locally to enable the search functionality.

3. How We Use Your Data

The data we collect is used solely for:

  • Authenticating with Spotify's API to enable search and playlist management features
  • Storing your search preferences and selected text locally in your browser

4. Data Storage

All data collected by Spotifinder is stored locally on your device using Chrome's built-in storage system (chrome.storage.local). We do not transmit, store, or have access to your data on any external servers.

5. Third-Party Services

Spotifinder integrates with the following third-party services:

  • Spotify API: Used for searching songs, managing playlists, and liking tracks. Your interaction with Spotify is governed by Spotify's Privacy Policy.
  • OpenRouter API (Optional): Used for AI-powered text parsing when enabled. If you choose to use this feature, your selected text may be sent to OpenRouter for processing. This is governed by OpenRouter's Privacy Policy.

6. Permissions

Spotifinder requires the following Chrome permissions:

  • activeTab: To access text you highlight on webpages
  • contextMenus: To add a right-click menu option
  • storage: To store OAuth tokens and preferences locally
  • identity: To handle OAuth authentication with Spotify

7. Data Sharing

We do not share, sell, or transmit your personal data to any third parties except as necessary to provide the extension's functionality (i.e., API calls to Spotify and optional OpenRouter services).

8. Your Rights

You have the right to:

  • Disconnect from Spotify at any time through the extension interface
  • Uninstall the extension, which will remove all locally stored data
  • Clear extension data through Chrome's extension management page

9. Security

We take data security seriously:

  • OAuth tokens are stored using Chrome's secure storage system
  • All API communications use HTTPS encryption
  • No sensitive data is transmitted to our servers (we don't operate any servers)

10. Changes to This Policy

We may update this privacy policy from time to time. We will notify users of significant changes by updating the "Last Updated" date. Continued use of the extension after changes constitutes acceptance of the updated policy.
